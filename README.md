# Juggernaut Marketing homepage redesign mockup

A static homepage redesign concept for Juggernaut Marketing, repositioning the brand around Oregon small businesses, transparent execution, and regional trust.

## Stack

- HTML
- CSS
- Vanilla JavaScript
- Static Vercel deployment

## Notes

- Branding cues were pulled from the current site at `gojuggernaut.com`
- Copy direction emphasizes Medford roots, Oregon small business positioning, and the Portland Trail Blazers small business partner point provided for the redesign brief
- The contact form uses a mailto flow so the mockup works immediately on static hosting
